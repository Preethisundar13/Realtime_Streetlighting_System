# Real-Time Street Lighting System Activated by Vehicle Motion Using 8051 Microcontroller

## Overview

This project is an energy-efficient smart street lighting system that automatically turns ON street lights only when vehicle motion is detected using IR sensors and an 8051 microcontroller.

## Components Used

* 8051 Microcontroller
* IR Obstacle Avoidance Sensors (6)
* Relay Module
* LED Lights
* Power Supply Module

## Features

* Motion-based street light activation
* Energy-saving automation
* Reduced operational cost
* Scalable for highways, streets, and parking lots
* Low maintenance

## Working Principle

The system uses IR sensors placed along the road to detect vehicle movement. Each sensor sends a signal to the 8051 microcontroller. Based on the detected input, the controller activates the corresponding relay and turns ON the connected LED (street light). Once the vehicle passes, the light turns OFF after a short delay.
